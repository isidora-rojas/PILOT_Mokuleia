# Mokuleia Reef - Wave Analysis
This repository contains analysis code, notebooks, and tools for processing wave-driven hydrodynamics over the Mokuleia reef. This project involves:
- Wave setup calculations
- IG/SS spectral decomposition
- Complex Demodulation
- Notebook-based exploratory analysis for meetings with Mark
- reproducible script in src/

## Repository Structure 
mokuleia/
├── Mark_meeting/        # exploratory + advisor-facing notebooks
├── src/               # core, modular scientific code
├── .gitignore
└── README.md
